# Why Journal

## Setup
1. Clone the repo.
2. Install node packages. We are using node version 14.15.4
```
npm install
```
3. Set up database and test db
```
createdb why-journal
```
```
createdb why-journal-test
```
4. Seed your database if you're developing.
```
npm run seed
```
5. Spin up a local instance.
```
npm run start-dev
```
